# Brand Brief System

How brand-architect builds a brand from one core idea. A brand is a feeling people recognize before they read a word. Build from one idea and let the name, palette, type, and voice all ladder back to it. References are curated and credited, never lifted. No em dashes, ever.

## The one idea

Before anything else, name the single idea the brand is built on, in one sentence. If you cannot say it in one sentence, keep working until you can. Every later choice is checked against this one idea.

## Moodboard structure

A moodboard is a curated set of references with stated rationale, sourced ethically. It never reproduces protected artwork, characters, or brand IP. Use image-sources for free, creditable images.

```
Moodboard: [brand name or working title]
The one idea: [one sentence]

For each reference (6 to 9 is plenty):
  - What it is and where it is from (credit the source)
  - Why it is here: the one quality we are pulling
  - What we are NOT taking from it

The thread: [the single feeling all references share]
```

Pull references from far outside the category. The freshest direction usually comes from somewhere unexpected.

## Brand attribute framework

Pick 3 to 5 attributes. For each, give the opposite it is chosen against, so the attribute means something.

```
Attribute: [word]    Not: [the rejected opposite]
How it shows up: [one concrete example in look, voice, or behavior]
```

## Naming method

1. Restate the one idea.
2. Generate from several angles: the feeling, the founder story, a coined word, a real word borrowed from another field, a sound.
3. Shortlist 5. For each: meaning, why it fits, the obvious risk.
4. Check the basics: easy to say, easy to spell, domain and handle plausible, no clash you can find in a quick search.
5. Recommend one, with the reason.

## Voice and tone capture

```
We sound: [3 words]
We never sound: [3 words]
A line we would say: [example]
A line we would never say: [example]
```

## One-page brand brief (the output)

```
Brand Brief: [name]
The one idea: [one sentence]
For: [who it is for and what they should feel]
Attributes: [3 to 5, each with its "not"]
Name: [recommended name, with the reason, plus the shortlist]
Voice: [we sound, we never sound]
Palette direction: [mood and rationale, not final hex unless known]
Type direction: [mood and rationale]
Moodboard: [link or summary]
Handoff notes: [what whoever executes the visuals needs to know]
```

Prefer one brave, ownable direction over three safe ones. Flag when a request would make the brand look like everyone else.
